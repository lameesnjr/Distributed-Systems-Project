# Distributed-Systems-Project
A cloud-based data processing service that enables users to upload datasets and perform distributed machine learning analytics using Apache Spark. This project demonstrates scalable data processing across 1, 2, 4, and 8 nodes with performance metrics analysis.

Technology Stack:
Frontend:     Streamlit
Processing:   PySpark, Spark MLlib
Storage:      Google Drive (Cloud Storage)
Deployment:   Google Colab / Cloud Services
Monitoring:   Ngrok for public access
Languages:    Python
